## Installation
```
npm install r-sentinel@latest
```

## Usage
> use Import
```
import Eval from 'r-sentinel'
```
> use Require
```
const Eval = require('r-sentinel');
```

```
Eval([
    yourFunction1();
    yourFunction2();
    yourFunction3();
    yourFunction4();
    yourFunction5();
]);
```
